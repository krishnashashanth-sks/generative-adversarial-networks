# GAN Zoo: Exploring Generative Adversarial Networks

Welcome to my repository dedicated to exploring and implementing various types of Generative Adversarial Networks (GANs). This project serves as a practical guide to understanding how different GAN architectures evolve and function.

## 📌 Project Overview
Generative Adversarial Networks are a powerful class of machine learning frameworks where two neural networks—a Generator and a Discriminator—compete against each other. This repository contains clean, single-file Python implementations of different GAN variants to study their mechanics, loss functions, and generation capabilities.
